# Agent Instructions

- This is an EmberJS project.
- Always use the latest Ember Polaris way of coding where applicable.
- Where appropriate, also use Ember Octane patterns.
- For frontend base components (mainly forms), use Frontile: https://frontile.dev/docs
- Styling uses TailwindCSS v4.
- Use ember-template-imports with `.gts` files for templates/components.
- Use the latest version of Warp Drive for querying data.
